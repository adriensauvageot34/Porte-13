Micro-checklist pour sécuriser tes étapes
  • Commence par verrouiller PATH/TARGET : choisis un chemin primaire dans PATH PICKER puis le couple primaire/secondaire dans TARGET PICKER avant d’ouvrir d’autres blocs.
  • Utilise les états bloquants : passe par le bloc [TEST:ALWAYS] pour cocher les tests obligatoires ; la barre de progression et le bouton Done se mettent à jour automatiquement.
  • Active les toggles “STOP” si doute : bascule [TOGGLE:SIG_OFF] ou [TOGGLE:CTRL_OFF] dans STATE avant de modifier des écoutes pour éviter un run incohérent.
  • Raccourcis et navigation : Ctrl+K ouvre la palette de commandes, et la navigation latérale suit ton scroll (highlight auto) pour revenir vite aux sections critiques.
  • Prends des notes rapides : journalise les essais dans JOURNAL ; tout est auto-sauvegardé (localStorage) pour ne pas reperdre le contexte.

TECH 7) TECH PACK — (Titre 4 replié)

A) TECH PACK — GÉNÉRAL P13

  • P13 = mise en scène : reverb/delay positionnent et hiérarchisent, ils ne décorent pas.
  • Un bon espace se ressent sans attirer l’attention ; si tu l’entends d’abord, il est trop présent. [REF:R3-10]
  • Principe : SEND/RETURN par défaut (dosage, cohérence, automation, EQ/duck sur l’effet sans toucher au dry). [REF:R3-13]
  • Insert uniquement si “design” assumé ; sinon risque de voler le message. [REF:R3-10]
  • Système minimal : 1 espace commun + 1 espace d’accent (throws).
  • Check 0 anti-biais : d’abord return level-match, ensuite paramètres. [REF:R3-05]
  • Filtrage returns : enlever bas, calmer haut ; sinon boue/fizz et fatigue. [REF:R3-06]
  • Si les “S” vivent dans l’ambiance : traiter le return plutôt que tuer la lead. [REF:R3-10]
  • Centre sacré : décorrélation qui casse le centre = FAIL (mono/instabilité). [REF:R3-04]
  • Tempo/groove : delay doit tomber “dans le film”. [REF:R3-10]
  • Cohérence : un seul monde par section ; changements accidentels = recadrage. [REF:R3-16]
  • Stutter delay : “scène/throw” = P13 ; “moteur rythmique/perf” = P15.
  • Si tu ne sais pas pourquoi tu ajoutes de l’espace, n’en ajoute pas. [REF:R3-09]
[PARAMS:HIDDEN] Repères concrets (retours FX)
  • Filtrage return : couper bas + calmer haut ; possible creux bas-médium si voile.
  • Pré-delay : colle vs séparation (plus séparé = plus audible).
  • Decay/feedback : débit rapide → plus court ; lent → peut respirer mais ducking devient prioritaire.
  • Diffusion/width : plus lisse vs plus granuleux ; lead tolère une largeur modérée (centre prioritaire).
  • Combo robuste : reverb plus centrée + delay plus large (profondeur + largeur sans flouter le centre).
  • Chaîne return : filtre → contrôle agressivité → contrôle dynamique/duck → (option) couleur → width/MS.

B) TECH PACK — Preset PATH:

Preset PATH:Ultra dry

  • Intention : impact, docu, texte au front.
  • Risque : plat / voix “dans la tête”.
  • Checkpoint : espace commun minimal, discret, mono safe.
  • Throws : rares, choisis.

Preset PATH:Tight room

  • Intention : “dans la pièce”, colle sans queue longue.
  • Risque : boxy/voile si ER trop présents.
  • Checkpoint : diction intacte en passage dense.

Preset PATH:Plate luxe

  • Intention : halo dense moderne, classy.
  • Risque : couvre la diction si non ducké/timé.
  • Checkpoint : consonnes + fins propres.

Preset PATH:Chamber

  • Intention : naturel + profondeur élégante.
  • Risque : flou si trop long.
  • Checkpoint : cohérence section + bas volume.

Preset PATH:Hall cinematic

  • Intention : grand écran / profondeur.
  • Risque : tails envahissantes, FX sujet, lead recule.
  • Checkpoint : centre sacré + mono.

Preset PATH:Slapback

  • Intention : épaisseur/vibe, présence.
  • Risque : répétition audible, groove perturbé.
  • Checkpoint : groove + diction.

Preset PATH:Delay rythmique

  • Intention : bounce, réponse rythmique.
  • Risque : pattern domine, masque texte si non contrôlé.
  • Checkpoint : passage dense + “phrase gagne”.

Preset PATH:Ping-pong wide

  • Intention : scène large, mouvement.
  • Risque : mono/corrélation, centre instable.
  • Checkpoint : mono + centre stable.

Preset PATH:Throws/Multi-tap

  • Intention : ponctuation (mot-clé, fin de phrase).
  • Risque : gimmick répétitif / incohérence monde.
  • Checkpoint : cohérence section + groove.

Preset PATH:Signature

  • Intention : reverse/gated/shimmer/blackhole = signature assumée.
  • Risque : fatigue, mono, FX sujet.
  • Checkpoint : ponctuel + secondaire + centre sacré. [REF:R3-10] [REF:R3-04]

Preset PATH:Convolution spéciale

  • Intention : IR atypiques pour un monde unique.
  • Risque : résonances IR / monde incohérent.
  • Checkpoint : cohérence section + passage dense.

C) TECH PACK — Preset TARGET:P13

  • RETURNS = point de contrôle : filtre + contrôle dynamique sur returns, pas sur le dry.
