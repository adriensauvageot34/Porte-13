3) [IF:FAIL] RED FLAGS (FAIL direct)

  • “Ça marche” seulement quand tu montes l’effet (return) → FAIL (biais de volume).
  • L’espace vole le premier plan (lead recule) → FAIL.
  • L’effet concurrence le texte (consonnes noyées) → FAIL.
  • Tails empilent les phrases (nuage permanent) → FAIL.
  • Le delay casse le groove → FAIL.
  • En mono, centre instable ou voix creuse → FAIL. [REF:R3-04]
  • Tu ne peux pas décrire simplement “où est” la voix → FAIL.
  • Tu utilises l’espace comme béquille pour “faire exister” la lead → FAIL → rollback Phase 2. [REF:R3-03]

7) TECH PACK (hors flux)

A) Général P13

  • P13 = mise en scène : reverb/delay positionnent et hiérarchisent, ils ne décorent pas.
  • Un bon espace se ressent sans attirer l’attention ; si tu l’entends d’abord, il est trop présent. [REF:R3-10]
  • Principe : SEND/RETURN par défaut (dosage, cohérence, automation, EQ/duck sur l’effet sans toucher au dry). [REF:R3-13]
  • Insert uniquement si design assumé ; sinon risque de voler le message. [REF:R3-10]
  • Système minimal : 1 espace commun + 1 espace d’accent (throws).
  • Check 0 anti-biais : return level-match puis paramètres. [REF:R3-05]
  • Filtrage returns : enlever bas, calmer haut ; sinon boue/fizz et fatigue. [REF:R3-06]
  • Centre sacré : décorrélation qui casse le centre = FAIL. [REF:R3-04]
  • Cohérence : un seul monde par section ; changement accidentel = recadrage. [REF:R3-16]
  • Stutter delay : scène/throw = P13 ; moteur rythmique = P15.

[PARAMS:HIDDEN] Repères concrets

  • Filtrage return : couper bas + calmer haut ; possible creux bas-médium si voile.
  • Pré-delay : colle vs séparation.
  • Decay/feedback : débit rapide → plus court ; lent → ducking prioritaire.
  • Diffusion/width : lisse vs granuleux ; lead tolère largeur modérée.
  • Combo robuste : reverb plus centrée + delay plus large.
  • Chaîne return type : filtre → contrôle agressivité → contrôle dynamique/duck → (option) couleur → width/MS.

B) Presets PATH

Les détails preset peuvent aussi être rendus depuis un registry plus tard. Pour l’instant, ceci sert de fiche.

  • Ultra dry : impact/docu — risque plat — checkpoint: espace minimal mono-safe.
  • Tight room : proche/réaliste — risque boxy/voile — checkpoint: diction en dense.
  • Plate luxe : halo dense — risque diction noyée — checkpoint: consonnes + fins.
  • Chamber : naturel/élégant — risque flou — checkpoint: cohérence + bas volume.
  • Hall cinematic : grand écran — risque tails envahissantes — checkpoint: centre sacré + mono.
  • Slapback : épaisseur/vibe — risque répétition — checkpoint: groove + diction.
  • Delay rythmique : bounce — risque pattern dominant — checkpoint: dense + phrase gagne.
  • Ping-pong wide : scène large — risque mono fragile — checkpoint: mono + centre stable.
  • Throws/Multi-tap : ponctuation — risque gimmick — checkpoint: cohérence + groove.
  • Signature : design assumé — risque fatigue/mono — checkpoint: ponctuel + centre sacré [REF:R3-10] [REF:R3-04]
  • Convolution spéciale : IR atypiques — risque résonances/incohérence — checkpoint: cohérence + dense.

C) Preset TARGET:P13

  • RETURNS : point de contrôle (filtre + contrôle dynamique sur returns, pas sur dry).
  • LEAD : sends dosés ; priorité au plan (devant), pas “beau”. [REF:R3-04]
