[IF:FAIL] FAIL — Red Flags + Actions
À ouvrir seulement si FAIL prouvé

RED FLAGS (FAIL direct)

  • “Ça marche” seulement quand tu montes l’effet (return) → FAIL (biais de volume).
  • L’espace vole le premier plan (lead recule) → FAIL.
  • L’effet concurrence le texte (consonnes noyées) → FAIL.
  • Tails empilent les phrases (nuage permanent) → FAIL.
  • Le delay casse le groove → FAIL.
  • En mono, centre instable ou voix creuse → FAIL. [REF:R3-04]
  • Tu ne peux pas décrire simplement “où est” la voix → FAIL.
  • Tu utilises l’espace comme béquille pour “faire exister” la lead → FAIL → rollback Phase 2. [REF:R3-03]

SYMPTÔMES SIGNATURE → ACTION (RUN)

SYMPTÔME : “mieux” seulement parce que plus fort (CHECK 0)
ACTION : level-match le return (1 levier)
RE-TEST : A/B bypass + Bas volume
STOP : à niveau égal seulement. [REF:R3-05]
SYMPTÔME : la lead recule dès que l’espace apparaît
ACTION : réduire l’espace sur la lead (1 levier)
RE-TEST : Lead au premier plan + Passage dense
STOP : centre sacré. [REF:R3-04]
SYMPTÔME : consonnes noyées quand l’effet apparaît
ACTION : faire se retirer l’effet pendant la diction (1 levier)
RE-TEST : Diction + Passage dense
STOP : phrase nette. [REF:R3-10]
SYMPTÔME : fins = nuage / phrases collées
ACTION : raccourcir l’espace (1 levier)
RE-TEST : Fins propres + Passage dense
SYMPTÔME : fatigue / “air” qui pique
ACTION : assombrir l’espace (1 levier)
RE-TEST : Bas volume + Plan
SYMPTÔME : boue/voile ajouté
ACTION : enlever le bas du return (1 levier)
RE-TEST : Passage dense + Bas volume
SYMPTÔME : taille d’espace change sans intention
ACTION : unifier l’espace par section (1 levier)
RE-TEST : Cohérence + SIG OFF/ON
STOP : “un monde” tient. [REF:R3-16] [REF:R3-02]
SYMPTÔME : l’écho parasite le flow
ACTION : simplifier les répétitions (1 levier)
RE-TEST : Groove + Passage dense

TECH TECH PACK — Général P13
Détails techniques hors flux
  • P13 = mise en scène : reverb/delay positionnent et hiérarchisent, ils ne décorent pas.
  • Un bon espace se ressent sans attirer l’attention ; si tu l’entends d’abord, il est trop présent. [REF:R3-10]
  • Principe : SEND/RETURN (dosage, cohérence, automation, EQ/duck sur l’effet sans toucher au dry). [REF:R3-13]
  • Check 0 anti-biais : return level-match d’abord, paramètres ensuite. [REF:R3-05]
  • Filtrage returns : enlever bas + calmer haut ; sinon boue/fizz et fatigue. [REF:R3-06]
  • Centre sacré : si décorrélation casse le centre = FAIL (mono/instabilité). [REF:R3-04]
  • Cohérence : un seul monde par section (sauf intention volontaire). [REF:R3-16]
  • Stutter : “scène/throw” = P13 ; “moteur rythmique/perf” = P15.
[PARAMS:HIDDEN] Repères concrets
  • Pré-delay : colle vs séparation (plus séparé = plus audible).
  • Decay/feedback : débit rapide → plus court ; lent → ducking prioritaire.
  • Chaîne return type : filtre → contrôle agressivité → duck → (option) couleur → width/MS.
